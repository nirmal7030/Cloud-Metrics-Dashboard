Metric Summary

Metrics Glossary

  • LCE – Latency & Connectivity Efficiency: network response reliability across requests.
  • PRT – Processing Response Time: average time for the system to respond to operations.
  • SMO – System Monitoring Overhead: CPU/memory usage due to security monitoring processes.
  • DEPT – Deployment Efficiency Time: average duration for CI/CD build & deployment.
  • CLBC – Cloud Load Balancer Consistency: load distribution and uptime efficiency metric.
